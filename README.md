# Proyecto de Práctica de Recursión

Este proyecto contiene implementaciones de dos algoritmos clásicos para practicar el concepto de recursión en JavaScript: Fibonacci y Merge Sort.

## Ejercicios

### 1. Fibonacci

El archivo `fibonacci.js` contiene dos funciones para generar la secuencia de Fibonacci:
- `fibs(n)`: Una implementación **iterativa**.
- `fibsRec(n)`: Una implementación **recursiva**.

Ambas funciones devuelven un array con los primeros `n` números de la secuencia.

### 2. Merge Sort

El archivo `mergeSort.js` contiene la función `mergeSort(array)` que ordena un array de números utilizando el algoritmo recursivo de Merge Sort.

## Cómo ejecutar el código

Ambos archivos están diseñados para ser ejecutados con Node.js desde la terminal.

Para probar el ejercicio de Fibonacci:
```bash
node fibonacci.js
```

Para probar el ejercicio de Merge Sort:
```bash
node mergeSort.js
```

## Requisitos

- **Node.js**: Necesario para ejecutar los archivos `.js` desde la terminal.
- **JavaScript Vanilla**: No se utilizan librerías ni frameworks externos.
